delugetools
==========

Some scripts for managing multiple deluge instances.

*deluge-cull*

Delete oldest torrents until free space threshold (in GB) is met:

`deluge-cull space --free 150`

Delete all torrents with "Unregistered torrent" error status:

`deluge-cull unreg`
